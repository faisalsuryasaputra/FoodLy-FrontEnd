# 🍽️ my-Foodly-app 🍽️

my-Foodly-app adalah aplikasi web berbasis React untuk mencari, melihat, dan mengelola resep makanan.

## 🚀 Tech Stack

- React 19
- React Router DOM 7
- Axios
- Bootstrap 5
- Lucide React
- React Icons

## 📋 Prerequisites

Sebelum menjalankan aplikasi, pastikan telah menginstal:

- Node.js (v18 atau lebih baru)
- npm

Cek instalasi:

```bash
node -v
npm -v
```

## 📥 Installation

### 1. Clone repository

```bash
git clone <repository-url>
```

### 2. Masuk ke folder project

```bash
cd my-Foodly-app
```

### 3. Install dependencies

```bash
npm install
```

atau install secara manual:

```bash
npm install axios bootstrap lucide-react react react-dom react-icons react-router-dom
```

## ▶️ Running the Application

Jalankan development server:

```bash
npm run dev
```

Aplikasi akan tersedia di:

```text
http://localhost:5173
```
